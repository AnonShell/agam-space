# Roadmap

Current focus and possible future features for Agam Space.

## Current focus: Bug fixes & improvements

Before adding major new features, the focus is on making the project stable and
reliable for production use.

**Priorities:**

- Comprehensive test coverage
- Bug fixes from early users
- Documentation improvements
- Performance optimization

## Possible features under consideration

No timeline commitments - features built as time permits.

- File sharing between users
- Public shareable links with expiry and password protection
- Search across files and folders
- Encrypted tags for organization
- S3-compatible storage backend support

## Feature requests

Have an idea for Agam Space? Open an issue on
[GitHub](https://github.com/agam-space/agam-space/issues) to discuss.

Keep in mind:

- Features must maintain zero-knowledge encryption
- Must be feasible for self-hosted deployments
- Should benefit the majority of users
